Fix #3: Upcoming Sessions Display Bug

The Problem:

When checking for upcoming sessions, the API was comparing the stored date (midnight) with the current datetime (including time). This caused today's entries to be filtered out!
‚ùå BEFORE (Broken)
if (upcoming === 'true') { filter.sessionDate = { $gte: new Date() }; filter.isProcessed = false; } // Compare: // Stored: 2026-02-15T00:00:00.000Z // With: 2026-02-15T11:34:00.000Z // Result: FILTERED OUT ‚ùå
‚úÖ AFTER (Fixed)
if (upcoming === 'true') { const today = new Date(); today.setHours(0, 0, 0, 0); filter.sessionDate = { $gte: today }; filter.isProcessed = false; } // Compare: // Stored: 2026-02-15T00:00:00.000Z // With: 2026-02-15T00:00:00.000Z // Result: INCLUDED ‚úÖ
